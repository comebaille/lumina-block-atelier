# Crédits visuels

LUMINA possède une identité, un logo, une interface et des matériaux de blocs originaux. Aucun asset extrait de Bloc Blast ou d’une autre licence n’est utilisé.

## Sanctuaire de cristal

- Origine : image générée pour ce projet avec l’outil intégré de génération d’images d’OpenAI.
- Usage : fond principal vertical du jeu.
- Source locale : `assets/art/crystal-sanctum.png`.
- Dérivé runtime : `assets/art/crystal-sanctum.webp`, encodé localement à qualité 86.
- Prompt complet et note de droits : `asset-manifest.json`.

## Identité et interface

- Logo LUMINA, icône PWA, matériaux de cristaux, panneaux, grille et VFX : créations originales du projet.
- Police : pile système locale uniquement ; aucune police distante ni hotlink.

La provenance et la couverture visuelle vérifiable sont centralisées dans `asset-manifest.json`.
